Installation

  1. Download or clone this repository.
  2. Copy the folder to C:\xampp\htdocs\.
  3. Open phpMyAdmin.
  4. Create/import the database using database.sql.
  5. Run setup.php once.
  6. Open login.php in your browser.

Local URL

http://localhost/full_login_logout_php_system/login.php
